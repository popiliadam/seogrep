Recipes map to the MCP prompts built into SeoGrep — multi-step flows you
can trigger with one plain-language request instead of naming each tool
yourself.

> **Beta note:** These are the three recipes available in beta. More land as
> the tool set grows.

## In this section

- [First site audit](/docs/recipes/first-site-audit) — the `new-site-audit`
  prompt, for a brand-new project.
- [Monthly SEO routine](/docs/recipes/monthly-seo-routine) — the
  `monthly-routine` prompt, for a recurring check-in.
- [Quick wins sprint](/docs/recipes/quick-wins-sprint) — the
  `quick-wins-sprint` prompt, for fast ranking improvements.

Each recipe assumes you've already added your personal MCP URL to your
client — see [Getting started](/docs/getting-started) if you haven't.
