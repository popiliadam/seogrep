This is the `quick-wins-sprint` prompt: a focused pass that looks for pages
close to ranking better, without running a full audit.

> **Beta note:** Tool names and thresholds may still change before launch.

## Goal

Find the smallest changes with the biggest ranking upside, fast.

## Example dialog

1. **You:** "Find quick wins for example.com."
2. **Ranklens:** runs `find_quick_wins`, then checks
   `detect_cannibalization` in case two pages compete for the same query.
3. **You:** "Any keyword gaps while you're at it?"
   **Ranklens:** runs `research_keywords` on the topics involved and folds
   the results into the same summary.

## What you get

A ranked list of pages worth a small edit, flagged cannibalization if any,
and a few related keyword ideas.
