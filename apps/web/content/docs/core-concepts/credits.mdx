title: Credits
description: How SeoGrep meters usage with credits, and where to check your balance.
---

Every analysis tool spends credits — crawling, auditing, keyword research,
and report generation each cost a set number of credits per run. Your AI
subscription is separate; credits only pay for the analysis work SeoGrep
does on its own servers.

> **Beta note:** Exact credit costs are draft numbers — see
> [/pricing](/pricing) for the current table.

## Checking your balance

Ask "What's my credit balance?" and SeoGrep runs `get_credit_balance`. The
dashboard shows the same number alongside your usage history.

## The cost guard

Before a run estimated above 200 credits, SeoGrep returns the estimate and
waits for you to confirm in chat instead of spending automatically.

## Running low

Trial accounts start with a one-time credit grant; paid plans renew monthly
and roll over unused credits for one month. See
[Billing & Credits](/docs/billing-and-credits) for plan-level detail.
