title: Ranklens Docs
description: What Ranklens is and how to get value from it in your first session.
---

Ranklens is a hosted MCP server for SEO analysis. You add your personal MCP URL
to the AI client you already use — Claude Desktop, claude.ai, Claude Code,
Cursor, or Windsurf — and run crawls, audits, and Search Console analysis by
asking in plain language. Credits pay for analysis work; your AI subscription
stays your own.
