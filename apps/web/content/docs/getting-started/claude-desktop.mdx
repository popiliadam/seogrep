Claude Desktop connects to Ranklens through a custom remote connector — no
local install or config file editing needed.

> **Beta note:** Remote MCP connectors are available on paid Claude plans.
> Your personal MCP URL comes from the Ranklens dashboard after your invite
> lands.

## Steps

1. Open Claude Desktop and go to **Settings → Connectors**.
2. Choose **Add custom connector**.
3. Paste your personal MCP URL (shown in these docs as `YOUR_MCP_URL`).
4. Save, then start a new chat and try "Crawl my site."

## If it doesn't connect

Restart Claude Desktop after adding the connector — new tools sometimes need
a fresh chat window. If the settings layout looks different from the steps
above, check Claude Desktop's own MCP documentation; connector menus have
moved before. See [Troubleshooting](/docs/troubleshooting) for more.
