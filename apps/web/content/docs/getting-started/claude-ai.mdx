claude.ai connects to SeoGrep the same way Claude Desktop does — through a
custom remote connector in your account settings.

> **Beta note:** Remote MCP connectors are available on paid Claude plans.
> Your personal MCP URL comes from the SeoGrep dashboard after your invite
> lands.

## Steps

1. Sign in at claude.ai and open **Settings → Connectors**.
2. Choose **Add custom connector**.
3. Paste your personal MCP URL (shown in these docs as `YOUR_MCP_URL`).
4. Save, then start a new chat and try "Crawl my site."

## If it doesn't connect

Refresh the page after adding the connector, and start a brand-new
conversation — existing chats don't pick up newly added tools. If the
settings layout looks different, check claude.ai's own MCP documentation.
See [Troubleshooting](/docs/troubleshooting) for more.
