title: Cursor

---

Cursor reads MCP servers from an `mcp.json` file — either per-project or
global to your user account.

> **Beta note:** Your personal MCP URL comes from the Ranklens dashboard
> after your invite lands.

## Steps

1. In your project, create `.cursor/mcp.json` (or edit your global Cursor
   MCP config to apply it to every project).
2. Add:

   ```json
   { "mcpServers": { "ranklens": { "url": "YOUR_MCP_URL" } } }
   ```

3. Save, then reload Cursor's MCP settings and try "Crawl my site" in chat.

## If it doesn't connect

Open Cursor's MCP settings panel and confirm "ranklens" shows a connected
status. If the field name above doesn't match what your version of Cursor
expects, check Cursor's own MCP documentation — this has changed before.
See [Troubleshooting](/docs/troubleshooting).
